# Codebase Second Brain (Mental Refinery)

Welcome to your command center.

## Structure

*   **[`00_Stream`](./00_Stream/)** (Fire)
    *   Start your day here. Journal, therapy, brain dump.
    *   Agent: **Morning Gardener** (Mindset hygiene).
*   **[`05_Reading_Room`](./05_Reading_Room/)** (Buffer)
    *   Raw materials waiting for processing.
    *   Agent: **Librarian** (Zero Inbox policy).
*   **[`10_Projects`](./10_Projects/)** (Ice)
    *   Work happens here. Tasks, plans, execution.
    *   Agent: **Project Manager** (Task decomposition, scope guard).
*   **[`20_Knowledge`](./20_Knowledge/)** (Resources)
    *   Learning happens here. Notes, articles, snippets.
    *   Agent: **Curator** (Connecting dots and learning).

## Quick Actions

1.  **New Day:** Enter `00_Stream`, create a file with today's date, and type `Start`.
2.  **Learning:** Paste an article into `05_Reading_Room` and ask for a summary.
3.  **Context:** Edit [ME.md](./ME.md) so Claude Code knows who you are.
